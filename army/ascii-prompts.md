# ASCII portraits — paste-able prompts for all five

Five prompts, one per army member, designed for **Nano Banana**
(Google Gemini Flash Image) with the SHADOW WOLF / JONINSCII reference
image already attached in the chat.

Each prompt is a single self-contained edit instruction: keep the
medium and frame from the reference, replace the figure, replace the
woven name with the character's real name, **replace the bottom
signature with the character's role label**.

## Role label table

The bottom signature in each piece is the character's role, in single-
word ALL CAPS to match the reference's "JONINSCII" rendering style.

| Character | Bottom signature |
|---|---|
| TENZO  | COMMANDER |
| KAJI   | ARTIFICER  |
| SENRI  | ARCHIVIST  |
| TETSU  | ENFORCER  |
| TAKA   | SENTINEL  |

Substitute the role into every `[ROLE — see role table above the prompt]`
slot below before sending. To override a provisional name (Senri /
Tetsu / Taka), edit the `REPLACE THE WOVEN "SHADOW WOLF" TEXT WITH:`
line — that's the only place the name appears.
