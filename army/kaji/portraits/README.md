# Kaji — portraits

ASCII-art emblems and any other visual identity pieces for Kaji himself.

## Naming convention

```
kaji-ascii-v1.png       (or .txt if pure-text ASCII)
kaji-ascii-v2.png
kaji-ascii.png          (canonical / chosen)
```

Kaji's ASCII pieces use **KAJI** as the woven name in the chest area
and **ARTIFICER** as the bottom signature (replacing the JONINSCII slot
in the reference). No separate callsign — the format settled on
real-name-on-figure + role-label-at-bottom.
