# System diagram suite — 5 paste-able prompts

A 5-panel set that describes the entire Tenzo Army architecture for a
fresh agent. Each prompt is self-contained — paste into a fresh ChatGPT
image-gen chat (NO reference image attached, per the medium-mismatch
lesson in `_ops/agent-chatgpt.md`).

All 5 share a unified style block (defined in each prompt) so they hang
together as a coherent suite when laid out side by side.

**Aspect ratio: 1:1 square for every image.**

When renders come back, save to:
`army/kaji/diagrams/system-suite-NN-<title>.png`

---

## SHARED STYLE BLOCK (already inlined in each prompt below — only edit if you want to change all 5 at once)

```
A clean infographic-style illustration in flat painterly aesthetic. NOT a
painting, NOT photographic, NOT 3D rendered. Think high-end product
architecture diagram crossed with stylized Asian-fantasy iconography
(simplified Avatar: The Last Airbender production design crossed with a
Stripe architecture diagram). 1:1 square composition.

PALETTE (strict): graphite (background), slate (mid-tones), deep teal
(accent), dull bronze (frames and watermarks), off-white (highlights),
thin gold (connection lines and labels). NO warm tones. NO orange, NO
red, NO firelight. NO glow effects beyond the thin gold accent lines.

FINISH: matte finish, restrained highlights, no glossy or wet-look
surfaces, clean confident linework. Labels in serif small caps, dull
gold. Generous negative space. No decorative elements outside the
diagram itself (no moons, no stars, no background symbols other than
the heraldic insignia watermark behind Tenzo when applicable).
```
